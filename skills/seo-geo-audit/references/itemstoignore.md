# Items to ignore

This is a list of issues and items to ignore in SEO audits. Do not include any of them when listing issues.

## 404s from `/email-protection`

`/email-protection` is a Cloudflare email obfuscation script and intentionally returns a 404 response to bots and crawlers. Ignore it when considering 404 errors.

## Title tag length

Title tags can be up to 150 characters long. Do not flag long title tags as an issue.
